# Smart To-Do List

![Smart To-Do List](screenshot.png)

## Overview

Smart To-Do List is a responsive task management web application designed to help users organize daily activities efficiently. The application allows users to create, manage, and track tasks while providing an engaging user experience through achievements, deadlines, and interactive elements.

## Features

✅ Add and delete tasks

✅ Mark tasks as completed

✅ Set task deadlines

✅ Weekly planning and task organization

✅ Achievement badge system

✅ Task completion rewards

✅ Interactive user interface

✅ Responsive design for different screen sizes

## Technologies Used

* HTML5
* CSS3
* JavaScript
